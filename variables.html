Exercises in variables in c
A variable is a space defined in memory that can contain a value of a particular type when the type of type is defined in a declaration of the variable. Each variable has a unique name and one value set. A variable stores data and its value can be determined and changed by the programmer (as it is). The variables must be declared in advance (thus allocating memory - the memory volume is measured in bytes).
Example of declaration of a variable:
int x;
The variable is, x of int type.
In defining the variables, it is important (not mandatory but highly recommended) to use meaningful names in order for our program to be readable and structured. Let's look at a few examples: Suppose our program summarizes the numbers from 1 to 10, then we will define a variable and its name sum. It is now clear that this variable will contain the sum of the numbers summarized at the end of the program. Or for example - a program that takes student names and grades, then we define two variables first name for saving the student's name and grade for saving his grade.
 Note: The variable name is usually limited to 32 characters. So if we declare a variable with 42 characters the compiler will ignore the last ten characters.
 Note: The c compiler is sensitive to changes in signal size so that a variable named Name, Name, NAME - are in fact three variables and not one and therefore the variables must be written.

Place of defined variables
Variables can be defined outside of functions or at the beginning of a block when declaring variables. Variables are divided into two types: Global and local variables.
Global Variables: Variables declared outside of a function constitute global variables relative to this function (since we have not yet discussed functions in c, global variables for now are variables that are declared outside of main ()). All of the functions that come after the statement recognize these variables, that is, a variable declared outside the function and it is recognized by the functions defined after its statement and until the end of the program.
Local Variables: Variables declared within the function and no external functions known to them, meaning "life span" is within the function itself (again - when discussing the functions we will see the uses).
Instead of showing realization of variables through functions, we can illustrate this by using blocks.
The reference of the variables to the same functions with respect to blocks is that a variable declared outside the block constitutes a global variable relative to it, whereas a declared variable inside the block constitutes a local variable relative to it.
To display the value of variables on the screen, a conversion is required, ie within the string we will not write the variable name (for example, int a = 7 the variable name is a and its value 7).

Types of variables
char - A variable for storing characters or symbols, occupying one byte in memory, where you can store integer values ​​from 128 to +127 characters and marks are all letters, punctuation, or other markings that can be generated by the computer.
int - Variable for storing an integer. Occupying two bytes of memory can store entire values ​​from 32768 to +32767.
float - a variable of real numbers up to 6 digits (stored by floating point method) The sub occupies four bytes in memory
double - Used to store real values ​​in larger domains, up to 10 digits, occupying eight bytes of memory.
long - can store entire values ​​in a range greater than int occupies four bytes in memory.
sing / unsing - Any numeric variable can be set to sing (with a +/- sign or or unsing (without sign) where sing indicates the storage of negative numbers and positive ones, while unsing specifies the storage of negative numbers, as a result of defining a variable as its positive range has increased . For example, sing char can store values ​​from the range (-128 +127) Conversely if we set the variable as unsing char values ​​from the range can be stored (0 255). The default defaults to an automatic variable.
Valid names for variables
A unique name must be assigned to each variable, thus allowing us to store a particular cell in a value memory and we can also refer to that cell to read the value. The variable cannot be named in any way that comes to our mind but there are rules for these variable names must start with a lowercase or a lower case (but we recommend using only the letters) nor can we name names that are reserved words like else if int etc. The rest of the variable name characters can be numbers, letters, and underscores, but be careful not to include additional characters such as a period, comma, and space.
