Arrays
So far we have dealt with simple actions like data entry from the user. Now we want to store and store multiple data of the same type - a data structure that allows us to do this is the array.
An array is a list of values, each value stored in a serial place in the array. The array basically constitutes a continuous set of memory cells for variables of the same type and in order to access each cell, its serial location must be specified in the array.
However, each organ in the array is a variable in itself and all actions learned on variables also apply to array organs (such as enlargement and self-reduction, etc.).
Example of using the system: If we want to get grades from 100 students, calculate the average grades, calculate the sum of the grades, and the standard deviation of the grades, then instead of declaring 100 variables, each of which will keep one grade, we will declare one set that will keep all 10 grades.
The size of the array must be specified in the statement (indicating its static nature - later chapters will be learned about dynamic data structure).

The array index starts from 0 to (lenth -1) so if we declare an array with 10 places then the index of the first place is 0 and the last 9.
Any out-of-cell contact can cause the program to crash, this error is not detected by the compiler.


Examples:
char arrayc [10]; - An array of 10-character characters (string)
int arrayi [20]; - An array of 20-place integers.
float arrayf [30]; - An array of real numbers with 30 places.


 Note:
An array of 10 int type spaces occupies 20 bytes of memory because int occupies 2 bytes. The third organ in the array, for example, is at the initial address of the array + 3 * 2.
The place of the i-organ in memory is: initial address + (the size of the type of house) * i.
That is, when an array is defined we do not know its initial address but we know that all of its organs are in succession one after the other. For example, if the first cell in the array is at 2000 and it is an array of integers (2 bytes in size) then the second organ will be found at the third 2002 address at 2004 and so on until the last one.
Ways to initialize an array
Each cell can be initialized separately:

int grades [5];
grades [0] = 90;
grades [1] = 80;
...

You can initialize the definition line by a list of variable values ​​within parentheses separated by a comma:
int grades [5] = {90, 80, ..};
Here, too, care should be taken not to list a number of organs that exceed the size we have defined.

 Note:
The size of the array cannot be specified if the array is initialized in this way (its size will be calculated automatically according to the number of data values).

Library string.h

A string handling library, we will see a number of functions in this directory and demonstrate their operation.
strlen (string) - A function that returns the number of characters in the string string (not including the character '\ 0').
strcpy (string1, string2) - The function accepts 2 string and copies the software of the second string for the first time (i.e. software of the string string1 after this command will be the content of string2).
For example:
char first [] = "ab";
char second [] = "cd";
strcpy (first, second);
first will contain the string "cd". strcat (string1, string2) - The function accepts 2 strings and concatenates the software of the second string for the first time (software of the string string1 after this command will be the original software and the string will be the string2 string).
For example :
char first [5] = "ab"
char second [] = "cd"
strcat (first, second)

Now it will first contain "abcd".
- strcmp (string1, string1) The function accepts 2 strings and returns 0 if the strings are the same, a positive value if string1> string2 and a negative value if string1 string2.
For example: strcmp ("abc", "abc") returns 0.
atoi (string) - The function receives a string string and returns its numeric value.
For example:
atoi ("12") returns the integer 12. itoa (num, string, baes) - The function accepts a number, a string, and a number representing a count basis and returns a numeric string that is the count at the base of the given number.
