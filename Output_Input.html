Library stdio
In previous chapters, we have superficially recognized the scanf input function and the printf output function - for standard input and output from the stdio directory.
In this section, we will introduce a number of functions of the stdio library and their structures:
printf and scan for built-in input and output. puts and gets for string input and output.
putchar and getchar for single characters.
That is, in this chapter we will see typical output input functions for specific types only.
Single character output - putchar () function
The putchar () function allows us to print a single character to the screen.
Let's illustrate this by an example:
putchar ('H');
This command will print the character H on the screen.
You can also enter a character value into a variable and enter the name of the variable into the putchar command, thus printing the value of the variable, for example:
char ch = 'a';
putchar (a);
Template of the putchar () function
putchar (char);
We will write a character between individual quotes, for example: 'H' but if we want to print the value of a character variable we will write between the round parentheses of the putchar function without single quotes or asci value even without quotes.
Single character input - getchar () function
The getchar () function allows a single character to be input from the standard input unit (keyboard).
We want to maintain the value of the character, so we make sure that the value returned by the getchar function is kept in a character variable.
For example:
char c;
c = getchar ();
So if the user pressed H then the variable c would store the value H.
Template of the () getchar function
char = getchar ();
We will use this command extensively because if we want to see the output of the program we will record the command at the end of the program and after that the program will stop and wait for user input. The program will not progress until it receives input from the user and on this occasion the user can read the output of the program and when finished press any key. We do not keep the character enclosed in the variable (because its value is irrelevant to us).
Structured output - function () printf
This function allows you to print several different variables by the same command (ie in one sentence).
The number of values ​​we want to print can vary, the command () printf supports changing the output structure according to our needs.
You can also print strings without using any variables. In this case, we will only write the string between quotation marks within the round brackets of the (printf) command.
Remember that different characters can be added to the command according to our needs - for example, adding "n \" allows you to move to a new line, etc.
Template of the command () printf:
printf (“….”, first_v, second_v,…, last_v);
The quotes of the () printf command must list the appropriate conversion template for each variable.



Built-In Input - function () scanf
This function allows for easy input of values. This function allows you to input several values ​​of different types (such as the printf () command allows you to print multiple variables of different types). The number of values ​​we want to absorb can vary, the scanf () command supports changing the input structure according to our needs.
Template of the command () scanf:
scanf ("……", & addres_of_first_v, & address_of_second_v, ..., & adderss_of_last_v);
The quotation marks of the (scanf) command must list the conversion templates of variables, that is, if you want to input a variable of an integer type, insert the quotation mark% d and the precursor name must include the address operator (&) and all variables. Care must be taken in writing the variables and conversion patterns.
In fact, by writing the address operator we make sure to enter the values ​​into the corresponding addresses of the respective variables.
 Note: If we forget the (&) address operator, we may crash the program.
Example:
long num_of_course;
float grade;
long id;
printf ("Enter your number of course id number and grade \ n");
scanf ("% ld% ld% f", & num_of_course, & id, & grade);
Let's summarize:
() getchar - Single character input function. Values ​​should not be entered between the parentheses of the function, the returned value (i.e. the user input) will be stored in the variable.
() putchar - Single character output function. The character brackets must be entered with single quotes or the name of a character variable or asci value.
() printf - Output function of several variables of different types and strings.

That is, all input and output operations are performed via a buffer and therefore unwanted data may be found in the buffer which may disrupt the program run and prevent it from clearing the buffer.
