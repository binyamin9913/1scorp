Control conditions
Their purpose is to allow us to change the program sequence, skip commands, and execute commands multiple times.
We divide the control sentences into three types:
Conditional execution commands:
The if and else commands
Switch and case commands
The operator?
Loops commands:
Commands for
The while command
The do while command
(And break and continue auxiliary commands)
Continue elsewhere in the program:
goto
Trial if
A control statement that allows us to make conditional execution of instructions or commands when a particular condition is checked and executed according to the value of the condition: true - represented by any number other than 0, and false - represented by the digit 0.
If statement allows us to branch out our program by logical terms as follows:
Template of the if command:
if (term)
{
    expression a;
}
else
{
    expression b;
}
That is, if the condition within the if statement is met (term value is "true" when the condition value is different from zero), follow the commands that appear between the curly brackets (expression a). If the condition is not met, follow the set of commands that appear between the two curly brackets (expression b).
 Note:
It is also possible without bracketed brackets but then only the first sentence after the condition will be performed, assuming the condition is true.

void main(){
int a;
printf("Enter a number between 0-100: ");
scanf("%d",&a);
if (a < 0)
printf("The number is not at the target");
else if (a < 10)
printf("The number is between 0-10");

printf("The number is between 30-40");

printf("The number is between 40-50");

Sentence swich
In the last example, we showed multiple choice by a complex if statement. Let us now see how it can be simplified and allowed for a more convenient and clear way of doing the following:
constant1- first constant.
constantn - the constant n.
switch (exp or variable)
{
    case constant1: command; break;
    case constant2: command; break;
    case constant3: command; break;
    case constantn: command; break;
    default: command;
}

switch receives a variable or expression and calculates its value. The resulting value is a constant (numeric or alphabetical value) and the command that corresponds to the calculated value is executed, that is, the program faces the appropriate case line and executes the commands that follow it until it reaches the break command.
The purpose of the break command is to prevent the following commands from being executed in the following cases.
If the value obtained from the switch statement does not match any of the cases, the instructions following the defaut command are followed.
 Note:
The default command does not have to appear but in this case if the constant calculated in the switch does not match one of the case-s, nothing will be done.
/*code*/#include
void main(){
char ch;
printf("Enter a char (between a-e): ");
scanf("%c", &ch);
switch(ch)
{
case 'a': printf("You have printed a");
case 'b': printf("You have printed b");
case 'c': printf("You have printed c");
case 'd': printf("You have printed d");
case 'e': printf("You have printed e");
default: printf("Your char is not between a-e");
}
}
}

Conditional expression?
The template of?
condition? exp1: exp2;
Before the? There is a condition and if the condition is met, the first expression must be performed. Otherwise (condition does not exist) the second expression (which appears immediately after the colon "":) must be performed.
The value obtained can also be placed in a variable or for printing (in the printf command).
In fact, this phrase can be translated into a sentence if:
if (condition)
     exp1;
else
     exp2;
Example:
int x, y, z;
if (x> y)
        z = x;
    else
        z = y;
Q: According to the conditional phrase? We will record this
    int x, y, z;
    z = (x> y)? x: y;
Thus, the variable z will have the maximum value of the variables x, y

Loop for
Loops allow instruction or block instruction in the program several times according to our needs.
When we want a block of commands or a single command to be executed multiple times we use the for loop.
The for loop is also called the incandescent loop.
The loop is made up of three parts:
1. Initialize variables.
2. Terms of performance.
3. Promoting variables.
Loop structure:
for (initialization; boolean condition; increment)
{
   . . .
}
The three parts of the loop are separated by them; (Semicolon).
In the first part variables are initialized, in the second the loop conditions are tested - where if the boolean value of this condition is true, the loop body will be executed and if we do not exit the loop and in the third part (performed after the loop body is executed) variables are promoted.
It should be noted that the boot is only done once at the loop entry, then the loop conditions are checked and if the value is verified, the loop body is followed, then the variables are promoted and the loop conditions are checked again. The routine will be performed until the loop conditions are not met and we will continue the program after the for loop.
 Note:
You can skip parentheses if the loop body consists of one line of code.


Omitting parts in the loop:
Sometimes we want to give up one of the loop components.
Any of the three loop parts (boot, loop conditions, promotion) can be empty.
For example:
int a = 5;v for (; a < 5; a ++)
{
    . . .
}
In this example, we skipped the variable boot since we initialized it.
int a;
for (a = 5; a < 5;)
{
    . . .
a ++;
}
In this example we omitted the promotion of the variable, the promotion done in the loop body.
The part of the loop condition cannot be omitted because we will then receive an infinite loop (although the problem can be overcome through a break, but this form is less acceptable).
/*code*/#include
void main(){
double i;
double result = 1;
printf ("Factorial function: \ n \ r");
for (i = 0; i < 10; i ++)
{
if (i! = 0)
result * = i;
printf ("% lf \ n \ r", result);
}
}
