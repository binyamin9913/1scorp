introduction
Define what a computer program is: Informally, this is a set of commands to execute for a computer. This series of commands starts from a predetermined place and ends with end commands written by the programmer. Define what is a c-language program: A c-language program is a program that collects commands that have a pre-agreed language - c language. This command collection is known to a programmer and to a specific software called "compiler for c language" or compiler c, which is responsible for translating the program from c language into a program containing commands known to the computer.


history The c language was developed in the early 1970s by bell, the original language was intended for the unix operating system but soon it expanded and developed a large number of versions. In 1983, ansi, the American Standards Institute, established a committee to set a standardized language standard to prevent the various differences between versions, this standard is called ansi c and all compilers are currently built to this standard.


Why c? C is a relatively small language and so easy to program. C is similar to pascal but has more power. Unix is ​​written in c language. Mobile C means that code written on one machine can easily pass to another. C allows the programmer to access the insides of the computer (bit level) C This is a modular language and is thus easy to understand and economical (tax functions can be called times with different arguments). C is the basis for the c ++ language Procedural programming means that the program is divided into several small code snippets called functions, each function addresses a specific problem. Contains a variety of program flow commands In principle, the computer executes commands in the order they are written, but the program flow can be controlled by commands and conditional execution of parts of the program or execution of certain sections numerous times thus saving code writing and storing it several times. By using loops. C contains a relatively small number of commands, so the compiler of the language is small and simple and the compilation results are effective.

Library functions
Library functions are "black boxes" that receive parameters, perform predefined operations, and return value. The parameters passed to the function are recorded in parentheses to the right of the function, the function is executed when the program "reaches" the function and the returned value can be treated as if it appears instead of the puncture after it has ended (for example, we saw in the first program use the printf function found in the stdio directory). We will discuss in more detail the functions in this chapter.
The directory we used in the first program is stdio. Each program I / O operation is performed using directory functions that handle I / O and for using these functions, the inclusion of the stdio library into the program is required.
We saw in the first program printf () printf (): This command is actually a set of commands for a computer that do not look like a programmer that causes everything that appears between the parentheses and the double quotation marks to be printed on the screen. There are extensions to the print command, these extensions will be written between quotation marks.
Main additions to printf (print command):
n \ - Move to a new line.
t \ - indents by several characters (depending on the default tab).
r \ - Move to a new row (just like pressing Enter).
"\- Apostrophes.
'\ - Gersh.
\\ - one reverse slash (\).
