Operands
Operands are variable or fixed. In the previous chapter we discussed the variables, in this chapter we will also deal with regulators.
Constants - variables that initialize to predetermined and known values ​​that do not change during program execution. Each constant is attributed to a specific variable type, for example pi is a double type constant and its value is known to all (3.1415). If we want to use our program with pi we will not want to allow another value to be pi (i.e. we will not allow ourselves to change the value of pi). To do this we define the variable as constant. To set the variable as constant, add the reserved word const before the type declaration.
Examples:
const double pi = 3.1415;
const int age = 65;
const long phone = 6543232;
Each constant can be forced to be int, double, float, etc. by adding an appropriate extension, and they:
If you want the constant to be of long type, add the extension L.
If you want the constant to be float type, add the extension F.
If you want the constant to be a double type, add the extension D.
If you want the constant to be of unsing type the suffix U.

Examples:
age = 65FL;
pi = 3.1415F;
In the first example we converted the variable type age from int to long.
In the second example, we converted the pi variable type from double to float. Note: Not all compilers support these extensions, support is only for new compilers.
It is also possible to place variables of different types but it is important to note when doing so that in some cases we may lose information. For example, if we put a float to int variable, we will lose the incomplete part of the number.
Operators
Placement operator - represented by a character = where the expression value on the right is placed in a variable on the left:
int a = 5;
int b = a;
First variable a will get the value 5 and then variable b will get the value of variable a i.e. the value of b after this statement will be 5.

Accounting Operators - Allows you to perform invoicing operations such as connection, subtraction, division, and modulo (residual):
a + b Connecting variables a and b.
a-b Subtracting the variable a in b.
a * b Multiplies variables a and b.
a% b Remainder of variable a divided by b.
